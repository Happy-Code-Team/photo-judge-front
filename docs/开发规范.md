## 命名规范

1. 组件名(中划线小写)
```text
login-form.vue
```

2. 变量名，函数名(驼峰)
```text
userInfo
```

3. 类名(大驼峰)
```text
UserInfo
```

4. 常量名(大写下划线)
```text
MAX_LENGTH
```

5. 文件夹名(中划线小写)
```text
components/user-info
```

6. 路由名(中划线小写)
```text
user-info
```

7. Css类名(中划线小写)
```text
.user-info {
  color: red;
}
```
